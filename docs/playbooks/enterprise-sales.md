# Playbook: Enterprise sales

**Goal**: move a Business/Enterprise conversation from demo to signed deal.

1. Lead with the persona-matched pain point from `docs/product/buyer-personas.md`, not a
   generic product tour.
2. Run the five-minute demo script (`docs/sales/demo-script.md`) live using the seeded
   captures closest to the prospect's own function (finance/ops, QA, support, IT,
   compliance).
3. Address security/compliance questions with the honest current-vs-roadmap framing in
   `/security` and `docs/sales/objection-handling.md` — do not overclaim certifications that
   don't exist yet.
4. Leave the one-page enterprise brief (`docs/sales/one-page-enterprise-brief.md`) as the
   forwardable artifact for internal champions to circulate.
5. Anchor pricing conversations on the Business tier; treat Enterprise as a custom
   negotiation once SSO/SCIM or dedicated audit-pipeline needs surface.
