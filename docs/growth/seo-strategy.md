# SEO strategy (concise)

**Pillar pages**: `/compare/*` (competitor-intent keywords: "scribe alternative", "loom
alternative", "windows steps recorder alternative") and `/solutions/*` (persona-intent
keywords: "SOP software for operations teams", "bug reporting tool for QA teams").

**Comparison pages are the highest-leverage SEO surface** for a new entrant — they capture
existing competitor search demand ("[competitor] alternative", "[competitor] vs") rather than
competing for generic, high-competition terms like "workflow software."

**Content cadence** (once resourced): one long-form post per month unpacking a friction
pattern from a real (anonymized) FlowLens capture — ties directly into the product's own
findings output, which doubles as content research.

**Technical basics already in place**: static generation for all marketing/solutions/compare
pages (`generateStaticParams`), per-page `metadata` titles. Not yet done: sitemap.xml,
structured data (Organization/SoftwareApplication schema), OG images per page.
