# Design inspiration notes

youmind.com was named in the build brief as a design-quality bar only (premium minimal SaaS,
spacious editorial layout, calm typography, soft gradients, refined cards, polished empty
states, restrained motion). No assets, code, layout, or exact wording were copied from it or
any other site; `docs/design/design-direction.md` documents the principles actually applied
in `apps/web` (spacious sections, single accent color, `.fl-card` component, restrained hover
transitions, light/dark aware CSS variables).
